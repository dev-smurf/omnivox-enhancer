# Omnivox Enhancer

This browser extension ports some Omnivox Mobile exclusive features.
## Features

- [X] Grading chart 
- [ ] Useful informations in footer
- [ ] Custom themes
- [X] Distracting element remover
- [ ] Grade faker 
- [ ] Mio enhancements
- [ ] Grade calculator
